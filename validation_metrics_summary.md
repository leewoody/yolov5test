## 📊 Precision 和 Recall 分析

### Precision (精確率)
- **定義**: 預測為正例中實際為正例的比例
- **公式**: TP / (TP + FP)
- **分析**: 
  - AR類別Precision最高 (100%)，但Recall很低
  - PR類別Precision較好 (73.68%)
  - MR類別Precision較低 (26.56%)

### Recall (召回率)
- **定義**: 實際正例中被正確預測的比例
- **公式**: TP / (TP + FN)
- **分析**:
  - PR類別Recall最高 (88.11%)
  - MR類別Recall中等 (59.65%)
  - AR類別Recall很低 (6.60%)
